# Project Principles

MovieDNA строится вокруг пяти ценностей.

1.
Каждая загадка должна вызывать эмоцию.

Не просто проверять память.

А дарить чувство открытия.

2.
Игрок никогда не должен чувствовать себя обманутым.

3.
Каждый элемент игры должен быть создан специально для проекта.

4.
Качество одной загадки важнее количества новых уровней.

5.
Любая новая функция должна делать ежедневную игру интереснее.

Если функция не улучшает основной игровой цикл —

она не должна быть реализована.
Player First

6. Любое спорное решение принимается

в пользу игрока,

а не разработчика.

7. Consistency

Все игровые механики должны использовать единые термины.

Например

Reveal Region

а не

Area

Zone

Mask

Polygon
