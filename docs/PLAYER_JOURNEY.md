# Player Journey

Продуктовый документ. Описывает путь игрока MovieDNA.

Любой новый PR должен улучшать этот путь.
Если PR не улучшает путь — сначала обсудить необходимость, не писать код.

Вопрос-фильтр: **поможет ли это игроку захотеть вернуться завтра?**

Правило краткости: если решение добавляет лишние действия перед следующей игрой — предпочесть более короткий путь.

---

## Полный путь

```text
Открыл сайт
↓
Понял правила
↓
Начал Daily
↓
Получил первую эмоцию («ага!» / напряжение угадывания)
↓
Прошёл Challenge
↓
Увидел результат (эмоция + Score + следующий шаг)
↓
Продолжил играть (Архив / вчера) ИЛИ открыл киномарафон
↓
Вернулся на следующий день
↓
Получил Streak
↓
Поделился результатом
```

---

## Что должно быть правдой на каждом шаге

### Открыл сайт
Игрок сразу понимает: это ежедневная игра про угадывание фильма по картинке-ДНК, не по кадру из фильма.

### Понял правила
До «Попробовать» видно: угадать фильм; области = подсказки и Score; ошибка открывает область; один Daily в день. Без длинного текста — за 5–10 секунд.

### Начал Daily
Один главный Challenge сегодня. Архив не подменяет Daily.

### Первая эмоция
Честные подсказки, момент узнавания. Игрок не проигрывает из‑за непонятной загадки.

### Прошёл Challenge
Победа или поражение — оба исхода закрыты. Нет тупика.

### Увидел результат
За 1–2 секунды ясно, что делать дальше. Порядок экрана:

1. Эмоция + Movie Score  
2. Главная кнопка следующего действия (без скролла): Archive → «Продолжить архив» сразу в следующий Challenge; Daily → вчерашний Challenge или «вернуться завтра»  
3. Компактная карточка киномарафона (если есть подборки) → отдельная страница `/movie/:slug/recommendations`  
4. Share  
5. Breakdown свёрнут  

Рекомендации фильмов не раздувают экран результата — это отдельная ценность продукта.

### Перешёл в Архив / Продолжил архив
После архивного Challenge один клик «Продолжить архив →» ведёт сразу в следующий непройденный день (`getNextArchiveChallenge`), без возврата в список. Если непройденных нет — сегодняшний Challenge или Архив.
Пройденные дни в архиве снова открываются: счёт, полная картинка и подборка — без повторной игры.

### Вернулся на следующий день
Daily доступен. Серия растёт при победе в Daily. Архив серию не ломает и не заменяет.

### Получил Streak
В профиле и на экране результата серия читается как причина вернуться.

### Поделился результатом
Хочется скриншот / share друзьям. Share не ломает опыт, если недоступен.
**Анти-спойлер:** в тексте шаринга нет названия фильма — получатель не должен узнать ответ до своей игры.

### Киномарафон (подборки)
Отдельная страница после результата: «что посмотреть» по категориям из `Movie.recommendations`. Только названия и подписи — без картинок Challenge и без ссылок «Играть» (иначе спойлер угадайки). Не обязательный шаг перед следующей игрой.

**Правило контента:** в рекомендации нельзя ставить фильмы из ближайшего окна будущих Daily Challenge (`scheduled` с `date > today`). Игрок не должен увидеть название фильма, который скоро угадает в игре.

## Roadmap по пути (кратко)

| PR | Шаг пути |
|----|----------|
| №1 Core Retention Loop | результат → архив / завтра / streak / профиль |
| №2 Эмоциональный результат | результат как награда + share-достижение |
| №3 Архив | архив = вторая сессия (CTA вчера + путь) |
| №4 Первый опыт | pre-start: показать суть за 5–10 сек |
| №5 Навигация и края | empty Daily, auth stub, mobile nav, share, human errors |
| №6 Movie Recommendations | данные подборок в Movie (категории) |
| №7 Better post-game | короткий результат + Continue Archive + киномарафон-страница |

Контент (14–21 Challenge вперёд) — постоянный процесс вне нумерации PR.
