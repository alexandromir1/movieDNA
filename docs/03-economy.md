# 03. Economy

## Purpose

Экономика MovieDNA определяет систему прогресса, мотивации и вознаграждения игрока.

Главная цель экономики — мотивировать игрока возвращаться каждый день не из-за страха потерять прогресс, а потому что он хочет стать лучше, проверить свои знания и открыть новую ежедневную загадку.

MovieDNA не использует агрессивные механики удержания, энергию, донатные преимущества или штрафы за пропуск дней.

---

# Core Principles

Экономика строится на следующих принципах:

- Skill First — вознаграждается мастерство, а не количество сыгранных дней.
- Fair Competition — новый игрок всегда может показать лучший результат дня.
- No Fear of Missing Out — игрок не должен чувствовать наказание за пропуск игры.
- Daily Motivation — ежедневная загадка является главным поводом вернуться.
- Transparency — игрок понимает, почему получил именно такой результат.

---

# Economy Overview

В игре существуют два независимых типа прогресса.

## 1. Movie Score

Результат прохождения конкретной ежедневной игры.

Используется для:

- сравнения игроков внутри одного Challenge;
- расчета процентиля;
- отображения личного результата.

Movie Score не переносится между ежедневными Challenge.

---

## 2. Player Profile

Долговременный прогресс игрока.

В профиле сохраняются:

- количество сыгранных игр;
- количество побед;
- процент побед;
- лучший стрик;
- текущий стрик;
- достижения;
- коллекция фильмов.

---

# Movie Score

Максимальный результат за прохождение одного Challenge составляет:

1200 Score

Movie Score складывается из четырех компонентов.

---

## Reveal Score

Каждый Challenge начинается с максимального значения **1000 Score**.

Во время прохождения игрок постепенно открывает Reveal Regions.

Каждое открытие автоматически уменьшает текущий Score.

Стоимость открытия Reveal Regions одинакова для всех Challenge и определяется только порядком открытия.

| Reveal Region | Penalty |
|---------------|--------:|
| Reveal Region 1 | -80 |
| Reveal Region 2 | -120 |
| Reveal Region 3 | -180 |
| Reveal Region 4 | -260 |
| Reveal Region 5 | -260 |
Данные значения являются частью игрового баланса MVP.*

После накопления статистики первых 100–200 опубликованных Challenge система будет пересмотрена на основе реальных данных игроков.

Reveal Regions всегда проектируются от наименее информативной к наиболее информативной.

Экономика не компенсирует плохой дизайн загадки. Если ранняя Reveal Region делает ответ очевидным, необходимо изменить саму загадку, а не стоимость открытия.

## Reveal Balance

Reveal Regions должны проектироваться таким образом, чтобы вероятность правильного ответа возрастала постепенно.

Рекомендуемая динамика:

| После открытия | Цель |
|---------------|------|
| До открытия | Лишь небольшая часть игроков может догадаться. |
| Reveal Region 1 | Появляются первые гипотезы. |
| Reveal Region 2 | Круг возможных фильмов начинает сужаться. |
| Reveal Region 3 | Большинство опытных игроков уже способны угадать фильм. |
| Reveal Region 4 | Ответ становится очевидным для большинства игроков. |
| Reveal Region 5 | Фильм должен быть узнаваем практически всеми, кто знаком с ним. |

Баланс достигается качеством Reveal Regions, а не изменением системы начисления очков.

## Time Bonus

Во время прохождения запускается скрытый таймер.

Игрок не видит отсчет времени.

Цель бонуса — поощрять уверенное прохождение, а не заставлять игрока спешить.

| Время | Bonus |
|--------|------:|
| до 20 секунд | +100 |
| 20–40 сек | +80 |
| 40–60 сек | +60 |
| 1–2 минуты | +40 |
| 2–4 минуты | +20 |
| более 4 минут | +0 |

Максимальный бонус составляет 100 очков.

---

## Guess Bonus

Начисляется за количество неправильных ответов.

| Ошибок | Bonus |
|---------|------:|
| 0 | +60 |
| 1 | +40 |
| 2 | +20 |
| 3+ | +0 |

---

## First Play Bonus

Если ежедневная игра была пройдена с первой попытки (без повторного прохождения после сброса данных), начисляется:

+40 Score

---

# Final Formula

Movie Score рассчитывается следующим образом.

Movie Score =
Reveal Score
+
Time Bonus
+
Guess Bonus
+
First Play Bonus

Максимальное значение:

1200 Score

---

# Percentile

После завершения игры игроку отображается его результат относительно остальных игроков данного Challenge.

Например:

Лучше, чем 74% игроков.

или

Вы вошли в Top 8% игроков этого Challenge.

Процентиль рассчитывается отдельно для каждого ежедневного Challenge и обновляется по мере поступления новых результатов.

В MVP используется только процентиль.

Глобальные рейтинги отсутствуют.
Percentile отображается только после завершения Challenge.

Показатель рассчитывается относительно всех завершенных прохождений данного Challenge.

По мере появления новых результатов процентиль может незначительно изменяться.

---

# Streak

Стрик показывает количество дней подряд, в которые игрок успешно завершил ежедневный Challenge.

Стрик не влияет на Movie Score.

Стрик не дает игровых преимуществ.

Стрик используется исключительно для открытия косметических наград.

Примеры наград:

- рамки профиля;
- бейджи;
- титулы;
- оформление профиля.

---

# Archive

Архив — catch-up для тех, кто зарегистрировался позже: можно один раз пройти старые Daily Challenge.

Правила:

- названия фильмов в списке скрыты;
- повторное прохождение после завершения недоступно;
- архивные прохождения не влияют на ежедневный рейтинг;
- не изменяют Movie Score опубликованного Challenge;
- не учитываются в процентиле.

---

# Statistics

В профиле отображается:

- сыграно игр;
- выиграно игр;
- процент побед;
- средний Movie Score;
- среднее количество открытых Reveal Regions;
- лучший результат;
- текущий стрик;
- лучший стрик.

Кроме абсолютных значений игрок может видеть собственную динамику.

Например:

- средний Movie Score за последние 30 дней;
- изменение среднего результата;
- количество Challenge, завершенных без открытия последней Reveal Region.

---

# Achievements

В MVP достижения отсутствуют.

Система достижений появится после релиза.

Примеры:

- первая победа;
- 10 побед;
- 100 побед;
- 30 дней подряд;
- угадал без открытия Reveal Regions;
- угадал после первой Reveal Region.

---

# Version 1.1

После релиза планируется добавить систему рейтинга игрока.

Player Rating будет рассчитываться отдельно от Movie Score и отражать общий уровень мастерства игрока.

Player Rating не зависит от количества сыгранных игр и постепенно изменяется после каждого Challenge.

Также планируется добавить:

- мировой рейтинг игроков;
- рейтинг друзей;
- сравнение по странам;
- расширенную аналитику профиля.

---

# Balance Philosophy

Экономика MovieDNA должна создавать чувство прогресса, а не чувство обязанности.

Игрок возвращается не потому, что боится потерять прогресс.

Игрок возвращается потому, что хочет стать лучше.

### Future Improvements (v1.1)

После получения достаточной статистики возможно внедрение адаптивной системы баланса.

Потенциальные направления:

- автоматическая корректировка стоимости Reveal Regions;
- анализ средней сложности каждого Challenge;
- динамическое распределение Score на основе статистики игроков.

Решение о внедрении принимается только после анализа реальных игровых данных.

## Player Progress

Movie Score остается единственной игровой оценкой.

Никакой валюты.

Никаких наград.

Никаких XP.
