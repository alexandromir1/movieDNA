# i18n (локализация)

## Архитектура

| Слой | Где | Как |
|------|-----|-----|
| UI-строки | `src/locales/ru.ts`, `en.ts` | `useTranslations()` / `t("nav.play")` |
| Контент фильмов | `data/movies/*.json` | `title: { ru, en }`, то же для rec titles/notes |
| Хелперы | `src/lib/i18n/` | `localize()`, `LocaleProvider`, `getRequestLocale()` |
| Поиск | `/api/movies/search?locale=` | ищет по названию **текущей** локали |

`acceptedAnswers` и `aliases` остаются плоскими `string[]` со всеми языками — это логика угадывания, не UI.

## Выбор языка

1. Cookie `moviedna-locale` (сервер)
2. Иначе `Accept-Language: en*` → `en`
3. Иначе `ru`
4. Клиент синхронизирует с `localStorage` и переключатель **RU | EN** в шапке

Смена языка — без перезагрузки (`LocaleProvider.setLocale`).

## Как добавить язык (например `fr`)

1. Расширь `Locale` в `src/lib/i18n/types.ts`: `"ru" | "en" | "fr"`
2. Добавь `src/locales/fr.ts` — зеркало ключей `ru.ts`
3. Подключи словарь в `LocaleProvider` (`dictionaries.fr = fr`)
4. В каждом movie JSON допиши `"fr": "..."` в `title` / rec fields
5. Прогони каталог поиска аналогично (или скрипт миграции)

Логику игры менять не нужно.

## Как добавить фильм с двумя языками

```json
{
  "id": "movie-example",
  "title": {
    "ru": "Матрица",
    "en": "The Matrix"
  },
  "year": 1999,
  "aliases": ["Matrix"],
  "recommendations": [
    {
      "title": {
        "ru": "💊 Если хочется ещё больше вопросов о реальности",
        "en": "💊 If you want more questions about reality"
      },
      "items": [
        {
          "movieId": "movie-inception",
          "note": {
            "ru": "любовь вопреки времени",
            "en": "love against time"
          }
        }
      ]
    }
  ]
}
```

В level `acceptedAnswers` перечисли RU + EN варианты ответа.

## Миграция контента

```bash
node scripts/migrate-i18n-content.mjs
```

Уже применена к playable movies + `search-catalog.json`.
